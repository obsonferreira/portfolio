## O que faz o meu projeto

### Módulo de controles

#### 1. Visão do usuário
- Criar contato  
- Editar contato  
- Validar dados  
- Salvar contatos  
- Exibir usuários  

#### 2. Visão técnica
- Controller de formulário (`inputDados.js`)  
- Validação de dados  
- Transformação de dados em objeto  
- Controller de tabela (`agendaDados.js`)  
- Geração de tabela com dados salvos  
- Filtro e organização da tabela
